Execution of the Program:
Requirements: Python 3.7 or above
1) Install jupyter notebook (if not installed) using the command:
   pip install notebook
2) Install all the python libraries used using the following commands:
   pip install numpy
   pip install pandas
   pip install seaborn
   pip install sklearn
   pip install matplotlib
   pip install plotly
   [ Internal libraries imported from seaborn are 'TfidVectorizer' and 'cosine_similarity' which has already been included in the package ]
2) Run jupyter notebook using the following command:
   jupyter notebook
3) Open 5G_B10_pgmfile.ipynb on jupyter notebook.
4) Download the datasets and put it in the same folder as the above file.
5) Run all the cells
